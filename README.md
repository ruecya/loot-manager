# 전리품 분배 매니저 (2레인 + 토큰 우선권)

## 기능 요약
- 명단(최대 50명) 등록/관리
  - 여러명 붙여넣기(줄바꿈/쉼표/탭)
  - 오늘 참여자만 ON (붙여넣기 기반)
  - 전체 ON / 전체 OFF
- 고정템 레인
  - 순번(포인터) 기반 지급/스킵
  - 우선권(토큰 1개) 적용: **토큰 이동(양보 보상)**, 포인터는 정상 진행
- 전리템 레인
  - 전리템 이름 입력 후 지급/스킵
  - 패스(토큰+1)로 토큰 생성
  - 우선권(토큰 1개) 적용: **토큰 이동(양보 보상)**, 포인터는 정상 진행
- 이력/백업
  - JSON 내보내기/가져오기
  - 로컬 저장(localStorage)

## 확정 룰
- 토큰 생성 루트: 전리템 차례에서 `패스(토큰+1)`
- 토큰 우선권: 토큰 1개 사용 시
  - 요청자(B) 토큰 -1
  - 원차례자(A) 토큰 +1
  - 아이템은 요청자(B) 수령
  - 순번 포인터는 **변경 없이 정상 진행(+1)**

## 로컬 실행
- `index.html`을 더블클릭해서 브라우저에서 열면 됩니다.
